Aula 1: Introdução aos conceitos básicos User Interface.

Visão Geral do Front-End

O front-end se refere à parte visual e interativa de um site ou aplicativo com a qual os usuários interagem. Exemplos incluem os elementos de interface em plataformas como o Instagram.

Tecnologias Principais

HTML: Define a estrutura e o conteúdo da página.
CSS: Gerencia o estilo visual e o layout.
JavaScript: Controla o comportamento e a interação da página.

Princípios de Design

O design é uma forma de comunicar ideias por meio de elementos gráficos, combinando forma e função para resolver problemas.

Conceitos de Usabilidade

A apresentação destaca cinco conceitos básicos essenciais para o design de uma interface de sucesso:

  • Aprendizagem: Os usuários conseguem realizar tarefas básicas na primeira tentativa?
  • Eficiência: Os usuários conseguem executar tarefas rapidamente após aprenderem como?
  • Memorabilidade: Os usuários conseguem se lembrar de como usar a interface após um tempo sem acessá-la?
  • Prevenção de Erros: O design ajuda os usuários a evitar erros ou a se recuperar deles facilmente?
  • Satisfação: A experiência é agradável e satisfatória para os usuários?

Exemplo

A usabilidade de um produto é comparada ao uso de uma balança simples, demonstrando conceitos como aprendizagem e eficiência através de cenários práticos.

Heurísticas de Design

A apresentação aborda as 10 Heurísticas de Usabilidade de Nielsen, como manter a consistência, oferecer ajuda e garantir o controle do usuário, evitando erros.

UX e UI

UX (Experiência do Usuário) e UI (Interface do Usuário) são destacadas como aspectos críticos do desenvolvimento front-end, com foco na interação do usuário e no layout visual, respectivamente.
