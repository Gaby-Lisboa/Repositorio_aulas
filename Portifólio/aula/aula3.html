Aula 3: Document Object Model (DOM)

O que é o DOM?

O DOM (Document Object Model) é uma interface de programação que permite que linguagens como JavaScript acessem e manipulem a estrutura de documentos HTML e XML como uma árvore de nós. Cada elemento, atributo e texto dentro do documento é representado como um nó, permitindo que programadores acessem e manipulem o conteúdo, a estrutura e o estilo da página da web de forma dinâmica.

Estrutura do DOM

A estrutura do DOM é baseada em uma hierarquia de nós, incluindo:

  • Node: A unidade básica que representa cada parte do documento.
  • Document Node: Representa o documento inteiro.
  • Element Nodes: Representam elementos HTML.
  • Text Nodes: Representam o texto dentro dos elementos.
  • Attribute Nodes: Representam os atributos de um elemento.

Manipulação do DOM com JavaScript

Com JavaScript, você pode acessar e manipular o DOM usando vários métodos disponíveis no objeto document. Alguns dos principais métodos são:

  • document.getElementById(id): Acessa um elemento pelo seu atributo id.
  • document.getElementsByClassName(className): Acessa elementos pelo nome da classe.
  • document.querySelector(selector): Acessa o primeiro elemento que corresponde a um seletor CSS.

Eventos e Manipulação do DOM

Eventos permitem a interação do usuário com a página. Com o DOM, você pode manipular eventos para criar interações dinâmicas, usando element.addEventListener(event, function) para adicionar manipuladores de eventos.

Vantagens do DOM

  • Interatividade: Permite criar experiências de usuário dinâmicas.
  • Manipulação em Tempo Real: Alterações podem ser feitas sem recarregar a página.
  • Facilidade de Acesso: A estrutura hierárquica torna fácil localizar e modificar elementos específicos.
