Aula 1 UI x UX.

13 de Agosto, 2024

Nesta aula, aprendemos a diferença entre UI (Interface de Usuário) e UX (Experiência de Usuário) no contexto do design de Front-End. Discutimos a importância de uma interface intuitiva e fácil de navegar, juntamente com os princípios de usabilidade que garantem a eficiência, memorização, prevenção de erros e satisfação do usuário. Exploramos também as Heurísticas de Nielsen, que fornecem orientações para melhorar a experiência do usuário, destacando pontos como consistência, controle do usuário e design minimalista. Compreender esses conceitos é essencial para criar interfaces que sejam tanto atraentes quanto funcionais.

Aula 2 Comandos básicos em JavaScript.

20 de Agosto, 2024

Nesta aula, focamos em aprender os comandos fundamentais do JavaScript, como variáveis, operadores, estruturas condicionais (if/else) e loops (for, while). Esses conceitos são essenciais para criar scripts que respondam às interações dos usuários e automatizem tarefas. Praticamos a criação de pequenas funções que realizam cálculos e manipulações de dados simples, fornecendo uma base sólida para entender o comportamento da linguagem JavaScript.

Aula 3 DOM em JavaScript.

27 de Agosto, 2024

Nesta aula, exploramos a manipulação do DOM (Document Object Model) usando JavaScript. Entender como o DOM funciona permite que desenvolvedores controlem o conteúdo e a estrutura de uma página web de forma dinâmica. Discutimos como selecionar e modificar elementos HTML, além de adicionar ou remover conteúdo com base nas interações do usuário, criando experiências mais interativas e responsivas.

Aula 4 Funções em JavaScript.

03 de Setembro, 2024

A quarta aula foi dedicada ao estudo das funções em JavaScript. Aprendemos como definir e invocar funções, além de entender a importância de usar funções para organizar e reutilizar código, tornando os programas mais modulares e fáceis de manter. Também exploramos o escopo de variáveis, parâmetros e valores de retorno, garantindo que o código seja eficiente e limpo. Esse conhecimento é fundamental para escrever código escalável e organizado em projetos maiores.

Aula 5 Arrays em JavaScript.

10 de Setembro, 2024

Nesta aula, exploramos os arrays em JavaScript. Aprendemos como armazenar e manipular coleções de dados, bem como acessar e modificar elementos dentro de um array. Discutimos métodos comuns como push, pop, shift e unshift, além de práticas para percorrer arrays usando loops. A manipulação de arrays é uma habilidade crucial para lidar com grandes volumes de dados em projetos de desenvolvimento.

Aula 6 POO em JavaScript.

A sexta aula introduziu os conceitos de Programação Orientada a Objetos (POO) em JavaScript. Discutimos a criação de objetos, classes, herança e encapsulamento, conceitos fundamentais para organizar o código de maneira eficiente. Entender POO é essencial para trabalhar com frameworks modernos de JavaScript e criar aplicações mais robustas e escaláveis.

Aula 7 Funções e DOM em JavaScript.

01 de Outubro, 2024

Na última aula, exploramos como manipular o DOM (Document Object Model) em conjunto com funções em JavaScript. Isso nos permitiu criar interações dinâmicas nas páginas web, como alterações no conteúdo e na aparência de elementos HTML com base nas ações dos usuários. Entender a interação entre JavaScript e o DOM é crucial para desenvolver páginas web interativas e responsivas.
