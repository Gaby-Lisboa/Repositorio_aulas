Aula 5: Arrays em JavaScript

Arrays em JavaScript

Os arrays em JavaScript são coleções de valores armazenados em uma única variável. Você pode manipular esses valores de várias formas, usando métodos como forEach(), map(), filter(), reduce(), e find().

  • forEach(): Itera sobre cada elemento do array e executa uma função.
  • map(): Cria um novo array com os resultados da função aplicada a cada elemento.
  • filter(): Filtra elementos com base em uma condição e retorna um novo array.
  • reduce(): Reduz os elementos de um array a um único valor.
  • find(): Localiza o primeiro elemento que satisfaz uma condição.

Manipulação de Arrays

Os métodos mais comuns de manipulação incluem adicionar/remover elementos do início ou fim de um array, remover por índice e muito mais:

  • push() e pop(): Adiciona e remove elementos no final do array.
  • unshift() e shift(): Adiciona e remove elementos no início do array.
  • splice(): Remove ou adiciona elementos em um índice específico.
