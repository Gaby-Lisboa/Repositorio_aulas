Aula 7: Funções e DOM em JavaScript

Funções em JavaScript

Funções são blocos de código reutilizáveis que executam tarefas específicas. Elas tornam o código mais modular e fácil de manter.

  • Funções Anônimas: Não possuem nome e são executadas no momento da sua criação.
  • Arrow Functions: Uma maneira mais concisa de escrever funções, melhorando a legibilidade do código.

Document Object Model (DOM)

O DOM é uma representação hierárquica dos elementos HTML da página, permitindo que o JavaScript manipule conteúdo, estilo e estrutura de forma dinâmica.

  • window: Representa a janela do navegador.
  • document: Representa a estrutura da página, como html, head, body, etc.
